# pyqt5_flashcards
The basic flashcards app made of Python and Qt5 <br>
Python version used here: Python 3.8.5 <br>
Qt version used here: PyQt5 (Qt bindings for Python) - as description <a href="https://pypi.org/project/PyQt5/">here</a><br>
<br>

<b>Note</b>: When running the scripts, please create new folders named "decks" and "img"
to run smoothly.
